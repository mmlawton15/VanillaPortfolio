My Work

This section contains my projects, group projects, and any future personal projects I work on (I have a few ideas up my sleeve). These apps are fully functioning, click the images to visit the page! If it's a backend project, you will be redirected to the Github Repo.

Horiseon

This was our first project of the bootcamp, where we essentially debugged the CSS and HTML files to make the website fully functional. This included making the links click to the correct sections and adding alt descriptions for every image.


Robot Gladiator

Robot Gladiator was our introduction to Javascript, and is a game that prompts the user to fight 3 different robot enemies. The users and robots were given health stats, the robots give different attack stats, and the game logic allowed the user to battle until they died or defeated all the robots. This gave us a foundation in Javascript basics, including primitive data types, functions, objects and control flow statements.

Code Quiz

The coding quiz is an app that quizzes players on a pre-identified set of questions. It runs on a timer that decreases with every incorrect answer, and with every correct answer you will get points. At the end, you are able to add your initials ad log your high score.

Work Scheduler

The Work Day Scheduler functions similarly to a calendar app, it allows the user to schedule their daily tasks by the hour and reflects the current date and time. The previous time is greyed out, the current time will be red, and the upcoming time will be green. This uses local storage to allow for tasks to persist after the browser has been closed. Users can edit, delete and create tasks on this page

Weather Dashboard

The Weather Dashboard was our introduction to Server-Side APIs. we were tasked with building a weather app, where the user could search for a city and get the daily forecast, as well as the 5 day forecast. The users recent searches were then logged as buttons on the far right for quick access.






Class Assignments

This section contains my projects that I worked on during class. These apps are fully functioning, click the images to visit the page! If it's a backend project, you will be redirected to the Github Repo.

Run Buddy

Run Buddy is a fitness app users can sign up for. When they sign up, the get assigned one of the pictured trainers and consult about their desired result. The point of the app is to come to Run Buddy to get a individualized consult and see results.

Password Generator

The Password Generator was our third graded Challenge of the bootcamp. It challenged us to use everything we had learned thus far in the bootcamp, including arrays, how to randomize arrays, global and local scope variables, and pseudocoding the logic behind what we wanted to do.

Taskinator

This was our third project we worked on during our bootcamp, and was our first exposure to Web APIs. In this challenge we learned how to access the DOM using different methods, add interactivity with HTML event listeners, handle JavaScript events using callback functions, use the DOM to read and create content to add/edit on the page, save data using localStorage and track technical debt in order to refactor code. It's modeled after a Kanban board.

Taskmaster Pro

This was an upgraded version of the Taskinator app we had worked on the prior week. It had an enhanced UI, with a new Delete All button, dates for tasks to be completed by, and tasks persisted using local Storage.

Git It Done

This class assignment was focused on an application that allowed users to search for open issues in Github. You would search by the username, and be presented with multiple different usernames that have different pending issues on their repository. You ca also filter by Javascript, HTML, or CSS issues based on what you want to work on.
