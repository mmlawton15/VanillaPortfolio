Gas Guru

Gas Guru was our first group project during our bootcamp. We were tasked with using 2 separate APIs and a CSS Framework other than bootstrap, along with building this application from scratch. The point of the app is to find the cheapest gas stations based on the city you enter. Based on the city, you got a forecast telling what the weather was and alerting you to drive safely. Our main struggle with this project was that we couldn't find a gas API that was free and allowed us to use the API an unlimited amount of times. When building this project, we used up the 10 free runs of the API. We tried multiple approaches to get around this with no luck. In the future we will most likely revisit this project and use an API that doesn't require a membership.
